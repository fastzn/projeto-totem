# 🖨️ Ficheiros de Impressão 3D (STL / 3MF)

Esta diretoria é reservada para as malhas 3D prontas para fatiamento utilizadas na fabricação das peças físicas do Projeto Totem. As peças foram projetadas considerando o preenchimento ideal para suportar o torque dos servomotores MG996R, utilizando materiais poliméricos como PLA ou ABS.
